## CHIP_IO.GPIO
Import the GPIO module as follows

  ```python
  import CHIP_IO.GPIO as GPIO
  ```

### toggle_debug()
Enable/Disable the Debug

* Parameters
  None

* Examples

  ```python
  GPIO.toggle_debug()
  ```
